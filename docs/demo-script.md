# Demo Script

Этот файл описывает сценарий для короткой демонстрации `Learning Mentor`.

Цель демо — показать, что приложение создаёт учебный workflow: тема, теория, задание, чат-наставник и сохранение прогресса.

## Подготовка

Для демонстрации используйте тестовый профиль и нейтральные учебные темы.

Рекомендуемая тема для демо:

```text
HTML: структура страницы
```

или:

```text
JavaScript: переменные и условия
```

## Короткое демо на 45–60 секунд

### 1. Запуск приложения

Показать стартовый экран приложения.

Ожидаемый результат:

- виден desktop-интерфейс;
- понятна основная цель приложения;
- пользователь видит учебный маршрут или список тем.

### 2. Выбор темы

Показать выбор темы из маршрута.

Ожидаемый результат:

- открывается объяснение;
- есть теория и примеры;
- интерфейс не выглядит как пустой чат.

### 3. Проверочное задание

Показать задание или тест по теме.

Ожидаемый результат:

- пользователь может проверить понимание;
- видно, что обучение состоит из действия, а не только чтения.

### 4. Чат-наставник

Показать вопрос в чат-наставник:

```text
Объясни проще и дай пример
```

Ожидаемый результат:

- наставник отвечает по текущей теме;
- объяснение связано с учебным контекстом.

### 5. Прогресс

Показать сохранение или изменение прогресса.

Ожидаемый результат:

- видно, что приложение запоминает состояние;
- пользователь может продолжить обучение позже.

## Расширенное демо на 2–3 минуты

Добавьте:

1. прохождение нескольких тем;
2. ошибочный ответ в проверочном задании;
3. возврат к слабой теме;
4. изменение настроек приложения;
5. запуск portable-версии.

## Что важно показать

В демо должны быть видны:

- desktop-приложение, а не только код;
- встроенный наставник;
- учебный маршрут;
- проверочное задание;
- прогресс;
- возможность portable-запуска.

## Что не показывать

Перед записью проверьте, что в кадр не попадают:

- личные настройки;
- реальные рабочие данные;
- локальные пути, если они раскрывают приватную информацию.

## Текст для описания демо

```text
Learning Mentor — desktop-приложение с AI-наставником для самостоятельного изучения fullstack-разработки. В демо показан учебный цикл: выбор темы, объяснение, проверочное задание, вопрос наставнику и сохранение прогресса.
```
