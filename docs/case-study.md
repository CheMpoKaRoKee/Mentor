# Case Study: Learning Mentor

## Контекст

Самостоятельное обучение разработке часто разваливается из-за отсутствия маршрута, обратной связи и регулярного повторения. Ученик может читать материалы, но не понимать, что делать дальше, как проверить себя и какие темы требуют возврата.

**Learning Mentor** — desktop-приложение, которое собирает учебный маршрут, объяснения, задания, чат-наставник и сохранение прогресса в одном интерфейсе.

## Проблема

Типичный процесс самостоятельного обучения:

1. ученик открывает разрозненные статьи и видео;
2. не всегда понимает порядок тем;
3. читает теорию без проверки;
4. быстро забывает пройденное;
5. теряет прогресс между сессиями;
6. не получает быструю обратную связь по непонятным местам.

Основные риски:

- нет единого маршрута;
- нет регулярной проверки знаний;
- сложно вернуться к слабым темам;
- мотивация падает из-за отсутствия видимого прогресса;
- AI используется хаотично, а не как встроенный учебный сценарий.

## Решение

Learning Mentor упаковывает AI-наставника в desktop-приложение.

Приложение помогает проходить короткий цикл обучения:

1. выбрать тему;
2. получить объяснение;
3. выполнить проверочное задание;
4. сохранить прогресс;
5. вернуться к сложным темам позже.

## Основной сценарий

```text
User opens desktop app
  -> selects learning topic
  -> receives theory and examples
  -> completes quiz or coding task
  -> asks mentor questions in chat
  -> progress is saved locally
  -> weak topics can be repeated later
```

## Что показывает проект

Проект демонстрирует не просто использование AI API, а продуктовый сценарий вокруг AI:

- AI встроен в учебный workflow;
- пользователь работает через desktop-интерфейс;
- прогресс сохраняется локально;
- приложение можно запустить как portable-версию;
- README описывает запуск, сборку, безопасность и roadmap.

## Интеграции и компоненты

| Компонент | Роль |
|---|---|
| Electron | desktop-оболочка |
| React | пользовательский интерфейс |
| Vite | сборка frontend-части |
| OpenRouter API | AI-наставник и генерация объяснений |
| Local storage / runtime data | настройки и прогресс пользователя |
| electron-builder | сборка desktop/portable-версии |

## Ценность как портфолио-проекта

Learning Mentor показывает навыки, важные для AI-интегратора:

- упаковать AI в понятный пользовательский сценарий;
- не ограничиваться чат-ботом;
- сделать интерфейс для регулярного использования;
- продумать хранение локальных данных;
- подготовить portable-сборку;
- оформить проект так, чтобы его можно было показать внешнему человеку.

## Ограничения MVP

- нужны реальные скриншоты интерфейса;
- нужно добавить GIF-демо прохождения урока;
- нужно оформить первый GitHub Release;
- желательно добавить тесты для логики прогресса;
- нужно расширить документацию по пользовательскому сценарию.

## Потенциальное развитие

- разные учебные маршруты;
- уровни сложности;
- экспорт прогресса;
- офлайн-доступ к уже сгенерированным урокам;
- режим повторения слабых тем;
- отдельные сценарии для подготовки к собеседованиям;
- шаблоны уроков под разные профессии и стеки.
