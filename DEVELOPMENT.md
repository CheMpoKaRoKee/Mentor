# Development Guide

## Назначение

`Learning Mentor` — desktop-приложение для самостоятельного обучения с AI-наставником, проверочными заданиями и сохранением прогресса.

Цель разработки — сохранять продуктовый сценарий: пользователь должен не просто общаться с AI, а проходить короткий учебный цикл.

## Локальный запуск

### 1. Установить зависимости

```bash
pnpm install
```

### 2. Создать локальный env-файл

```bash
cp .env.example .env
```

На Windows можно создать `.env` вручную или через PowerShell:

```powershell
Copy-Item .env.example .env
```

### 3. Запустить dev-режим

```bash
pnpm run dev
```

### 4. Собрать приложение

```bash
pnpm run build
```

Сборка создаётся в `dist/`. Эта папка не хранится в репозитории.

## Основные части проекта

Рекомендуемое разделение ответственности:

- `src/main/` — главный процесс Electron, работа с настройками, локальными данными и AI API;
- `src/preload/` — безопасный мост между Electron и React;
- `src/renderer/` — интерфейс приложения, экраны, компоненты и пользовательский сценарий;
- `Portable/` — локальная portable-сборка и служебные файлы для релизов;
- `docs/` — портфолио-документация, кейс и демо-сценарии.

## Smoke test после изменений

После изменения кода проверьте:

1. приложение запускается через `pnpm run dev`;
2. открывается стартовый экран;
3. можно выбрать тему;
4. отображается объяснение;
5. работает проверочное задание;
6. чат-наставник отвечает в контексте темы;
7. прогресс сохраняется;
8. сборка `pnpm run build` проходит без ошибок.

## Правила изменения проекта

При изменении пользовательского сценария обновляйте:

- README;
- `docs/case-study.md`;
- `docs/demo-script.md`;
- `docs/screenshots.md`, если изменились экраны.

При изменении настроек обновляйте:

- `.env.example`;
- README;
- `SECURITY.md`, если меняется работа с локальными данными.

## Portable-режим

Перед публикацией portable-сборки проверьте:

- пользовательские данные не попадают в архив;
- локальный env-файл не попадает в архив;
- временные файлы и логи не попадают в архив;
- README содержит актуальную инструкцию запуска.
