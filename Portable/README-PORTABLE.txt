Learning Mentor Portable

Запуск:
1. Откройте файл Learning Mentor.exe.
2. При первом запуске укажите OpenRouter API key в настройках.
3. Рабочие данные будут созданы в папке data рядом с программой.

Как получить OpenRouter API key:
1. Откройте https://openrouter.ai/keys.
2. Войдите в аккаунт OpenRouter.
3. Создайте новый ключ.
4. Скопируйте его и сохраните в настройках приложения.

Как задать ключ через .env:
1. Создайте файл .env рядом с Learning Mentor.exe.
2. Добавьте в него строку:

OPENROUTER_API_KEY=ваш_ключ_openrouter

3. Перезапустите приложение.

Что важно:
- portable.flag включает хранение данных внутри этой папки, а не в AppData;
- прогресс сохраняется в data\progress.json;
- настройки сохраняются в data\settings.json;
- папку Portable можно перенести на другой диск или флешку.

Перед публикацией portable-архива убедитесь, что папка data отсутствует
или не содержит личных данных, ключей, cookies, логов и кэша.
